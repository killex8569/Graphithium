# Graphithium

Graphithium is an open-source project, I'm currently working alone on it so be patient. Graphithium is a web app developed using Vue.js and Node.js, it is essentially a web app-based alternative to Obsidian !

# Installation

to strat the projet --> `rpm run dev`


# Features

when your **account** is set up, you can log in and use the following features. You can create your own "vault" and give it a **name**, you can place your vault anywhere you want. After creating/loading your vault, the app allows you to create files, to modify them and delete your file or your folder. Next we can add

## Account

First you need to create an account (special dedicated page), on your profil, you can add :
- Name
- Pseudo
- Description
- Profile picture
- Status 
- Online visibility (Online, Idle, Do not Disturb, Invisible)
- Localisation (Country/State)

A section below will be implemented which allow you to change your password (your password, new password, verify password)
